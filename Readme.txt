Project: Personal Portfolio Website

This website is a personal portfolio to showcase my web development skills, projects, and experience. It includes:

- Home: Brief introduction and profile photo
- Projects: Gallery of my main works
- Skills: List of technical skills
- Services: Overview of services offered
- Contact: Form to reach me
- About: More about my career and education

Built with HTML, CSS, JavaScript, Bootstrap, and Bootstrap Icons.
مشروع: موقع بورتفوليو شخصي

هذا الموقع هو عبارة عن بورتفوليو شخصي يهدف إلى عرض أعمالي ومهاراتي وخبراتي في مجال تطوير الويب. يحتوي الموقع على عدة صفحات مثل:

- الصفحة الرئيسية: نبذة مختصرة عني وصورة شخصية.
- صفحة المشاريع: عرض لأبرز المشاريع التي عملت عليها مع صور وروابط.
- صفحة المهارات: قائمة بالمهارات التقنية التي أتقنها.
- صفحة الخدمات: شرح للخدمات التي أقدمها.
- صفحة التواصل: نموذج للتواصل معي عبر البريد الإلكتروني أو وسائل التواصل الاجتماعي.
- صفحة حول: معلومات إضافية عن مسيرتي المهنية والتعليمية.

تم بناء الموقع باستخدام HTML و CSS و JavaScript، مع الاستفادة من مكتبة Bootstrap و Bootstrap Icons لتحسين التصميم والاستجابة.

جميع الصور والملفات المستخدمة في الموقع محفوظة في مجلد "assets".

لأي استفسار أو اقتراح، يرجى التواصل معي عبر صفحة التواصل في الموقع.
